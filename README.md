# Workcloud Plus

Chrome Extension that enhances Workcloud

## Prerequisites

* [node + npm](https://nodejs.org/)

## Setup

```bash
npm i
```

## Build

```bash
npm run build
```

## Build in watch mode

```bash
npm run watch
```

## Load extension to Chrome

Load `dist` directory.
